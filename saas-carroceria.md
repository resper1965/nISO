# nISO | SaaS Transformation Tasks

## Goal
Transformar a interface contemplativa em uma ferramenta operacional SaaS para auditoria ISO.

## Tasks
- [ ] **Task 1: Compact Sidebar & Utility Nav** -> Reorganizar menu lateral para fluxos de trabalho (Discovery, Evidence, Compliance, Docs).
- [ ] **Task 2: Actionable Dashboard Widgets** -> Substituir textos grandes por widgets de status (Readiness Progress, Task Queue, PII Scanner Status).
- [ ] **Task 3: Control Detail View** -> Criar tela de detalhe de controle com histórico e botão de upload de evidência.
- [ ] **Task 4: Evidence File Manager** -> Criar interface para listar, baixar e validar integridade (SHA-256) de arquivos no R2.
- [ ] **Task 5: Policy Generator UI** -> Integrar o serviço de geração de IA com um botão "Generate Draft" direto no SoA.

## Done When
- [ ] O Dashboard mostra métricas de ação reais.
- [ ] É possível navegar entre "Discovery" e "Evidence" com fluxos claros.
- [ ] O design mantém a brand ness. mas com densidade de informação SaaS.
